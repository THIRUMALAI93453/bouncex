# 🎮 Bounce X

**Bounce X** is a fast-paced **web-based arcade survival game** built using HTML5, JavaScript, and modern web technologies. The game focuses on **reflex-based gameplay**, where players control a continuously bouncing ball and survive by landing on dynamically generated tiles.

The experience is enhanced with **adaptive difficulty, combo systems, power-ups, and real-time visual effects**, making it both engaging and highly replayable.

---

## 🚀 Key Features

* 🎯 Reflex-based survival gameplay
* ⚡ Adaptive difficulty system
* 🔥 Combo streak mechanics (x3, x5, x10 effects)
* 🛡️ Power-ups (Shield, Magnet, Speed Boost)
* 💎 Diamond collection & reward system
* 🌈 Dynamic color-changing environments
* 💥 Camera shake & slow-motion effects
* 🎮 Smooth physics-based controls
* 📅 Daily challenge mode (seed-based gameplay)
* 👕 Unlockable ball skins system

---

## 🧠 Concept

Bounce X is designed as more than just a game — it acts as a **cognitive reflex training system**, improving player reaction speed, timing accuracy, and decision-making under pressure.

---

## 🛠️ Tech Stack

* HTML5
* JavaScript / TypeScript
* WebGL / Three.js (for visual effects)
* CSS Animations
* LocalStorage (for saving progress & skins)
